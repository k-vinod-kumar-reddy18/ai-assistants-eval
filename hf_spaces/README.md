# Qwen2.5 OSS Assistant

Personal AI assistant built with Qwen2.5-0.5B-Instruct, featuring:

- **Multi-turn memory** — last 5 conversation turns as context
- **Guardrails** — input/output safety layer blocking harmful requests
- **Clean Gradio UI** — instant usability, no setup required
